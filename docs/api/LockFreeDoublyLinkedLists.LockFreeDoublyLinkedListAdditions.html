CompareExchangeValueIf<T>(ILockFreeDoublyLinkedListNode<T>, Func<T, Tuple<T>>)

Exchanges the value of a LockFreeLinkedList node with a new Value generated from the old value if and only if newValue for the old value is not null. If not, nothing is performed and false within the result Tuple is returned.
